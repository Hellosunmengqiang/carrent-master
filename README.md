# carrent
SpringBoot共享汽车系统-Car Rental Management On Spring Boot

* 涉及通信方面的功能
* 前端:js+jq+bootstrap+模板
* spring boot + mybatis
* mysql
* 邮箱验证码注册
* echarts data view
* alipay(支付宝支付) sandbox
* dropzone pictures upload(多图片上传)
* regist with sms number (电话短信注册)
* use Gentelella Admin for admin pages
* and more
